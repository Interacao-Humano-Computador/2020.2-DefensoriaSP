## Rodar localmente

### Dependências

- [docker](https://docs.docker.com/get-docker/)
- [docker-compose](https://docs.docker.com/get-docker/)

### Comandos

- Para compilar:
<pre><code>docker-compose build</code></pre>

- Para rodar:
<pre><code>docker-compose up</code></pre>

- Acesse a url local http://0.0.0.0:8000/
